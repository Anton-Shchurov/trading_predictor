# Конфигурация проекта

Эта папка содержит конфигурационные файлы для проекта.

## Настройка OANDA API

1. Скопируйте файл `oanda_config.example.yml` в `oanda_config.yml`:
   ```bash
   cp oanda_config.example.yml oanda_config.yml
   ```

2. Откройте файл `oanda_config.yml` и замените `YOUR_OANDA_API_TOKEN_HERE` на ваш реальный API токен от OANDA.

3. При необходимости измените другие настройки (URL, инструмент, гранулярность данных).

## Получение API токена OANDA

1. Зарегистрируйтесь на [OANDA](https://www.oanda.com/demo-account/) для получения демо-аккаунта
2. Войдите в личный кабинет
3. Перейдите в раздел "Manage API Access"
4. Создайте новый API токен
5. Скопируйте токен в файл `oanda_config.py`

## Безопасность

⚠️ **ВАЖНО**: Файл `oanda_config.yml` содержит чувствительные данные и НЕ должен попадать в систему контроля версий. Он добавлен в `.gitignore` для предотвращения случайной загрузки.

## Требования

Для работы с YAML конфигурацией необходимо установить библиотеку PyYAML:
```bash
pip install PyYAML
```

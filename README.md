# TradingPredictor

Система прогнозирования валютных курсов с использованием машинного обучения и глубокого обучения.

## Структура и описание проекта

```
TradingPredictor/
│
├── 01_data/                        # Исторические и обработанные данные
│   ├── raw/                     # Сырые CSV/Parquet от различных источников
│   ├── interim/                 # Промежуточные данные и автоматические EDA отчеты
│   │   ├── eda_report_*.md      # Markdown отчеты с полным анализом данных
│   │   └── images/              # Изображения графиков высокого качества
│   └── processed/               # Финальные данные с фичами и лейблами (готовые с признаками и метками для обучения моделей)
│
├── 02_notebooks/                  # Jupyter / Colab / Markdown-исследования
│   ├── eda/                    # Исследовательский анализ данных (EDA)
│   ├── feature_engineering/    # Прототипы признаков (разработка и тестирование новых признаков)
│   └── experiments/            # Модели и сравнения (эксперименты с моделями и их сравнение)
│
├── 03_src/                        # Исходный код
│   ├── data/                   # Сбор и подготовка данных (модули для сбора и предварительной обработки данных)
│   ├── features/               # Генерация фич (генерация признаков и feature engineering)
│   ├── models/                 # ML и DL-модели (реализация ML и DL моделей)
│   ├── backtest/               # Стратегии и симуляторы (бэктестинг торговых стратегий)
│   ├── trading/                # Связь с брокером (интеграция с торговыми платформами и брокерами)
│   ├── dashboard/              # Веб-интерфейс (веб-интерфейс для мониторинга и управления)
│   └── monitoring/             # Логи, алерты, MLflow (система логирования, алертов и отслеживания метрик)
│
├── 04_configs/                   # Конфигурации моделей и пайплайнов (конфигурационные файлы для моделей, пайплайнов и параметров системы)
├── 05_logs/                      # Логи трейдинга и моделей (логи работы системы, торговых операций и моделей)
├── 06_reports/                   # Отчёты, графики, метрики (отчёты, графики производительности и аналитические материалы)
├── 07_common_data/               # Общие данные (API ключи, настройки) (общие данные проекта: API ключи, настройки подключений, справочная информация)
├── 08_tests/                     # Unit- и integration-тесты для кода и моделей
│
├── README.md

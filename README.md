# TradingPredictor

Система прогнозирования валютных курсов с использованием машинного обучения и глубокого обучения.

## Структура и описание проекта

```
TradingPredictor/
│
├── 01_data/                        # Хранение данных на всех этапах
│   ├── raw/                        # Исходные данные (например, CSV от OANDA/Dukascopy)
│   ├── interim/                    # Промежуточные данные и автоматические EDA отчеты
│   │   ├── eda_report_*.md         # Markdown отчеты с полным анализом данных
│   │   └── images/                 # Изображения графиков высокого качества
│   └── processed/                  # Готовые датасеты для обучения (parquet)
│
├── 02_notebooks/                   # Jupyter ноутбуки для экспериментов
│   ├── eda/                        # Исследовательский анализ данных (EDA)
│   ├── feature_engineering/        # Прототипы признаков (разработка и тестирование)
│   └── experiments/                # Эксперименты с моделями и их сравнение
│
├── 03_src/                         # Исходный код проекта
│   ├── data/                       # [Планируется] Загрузка и обработка данных
│   ├── features/                   # Логика генерации признаков
│   │   ├── feature_pipeline.py     # Основной пайплайн создания фич
│   │   ├── lag_features.py         # Создание лаговых переменных
│   │   ├── statistical_features.py # Скользящие средние и статистики
│   │   ├── technical_indicators.py # Технические индикаторы
│   │   └── target_labels.py        # Генерация целевых меток
│   ├── models/                     # Обучение и оценка моделей
│   │   ├── modeling_pipeline.py    # Пайплайн обучения
│   │   ├── feature_importance.py   # Анализ важности признаков
│   │   └── hyperparameter_tuning.py # Подбор гиперпараметров (Optuna)
│   ├── reporting/                  # Генерация отчетов (EDA)
│   ├── backtest/                   # [Планируется] Бэктестинг торговых стратегий
│   ├── trading/                    # [Планируется] Связь с брокером
│   ├── dashboard/                  # [Планируется] Веб-интерфейс
│   └── monitoring/                 # [Планируется] Логи, алерты, MLflow
│
├── 04_configs/                     # Конфигурационные файлы (YAML)
│   ├── feature_engineering.yml     # Настройки генерации фич (периоды, окна)
│   ├── models.yml                  # Параметры моделей (CatBoost, Ridge, и др.)
│   ├── splits.yml                  # Настройки разбиения данных (Train/Val/Test)
│   └── oanda_config.yml            # Доступы к API брокера
│
├── 05_logs/                        # Логи работы системы, трейдинга и моделей
├── 06_reports/                     # Отчёты, графики производительности
├── 07_common_data/                 # Результаты экспериментов и общие данные
├── 08_tests/                       # Unit- и integration-тесты
│
├── requirements.txt                # Список зависимостей Python
└── README.md
```
